### Week 4 Objectives:
* Gain an in-depth understanding of Amazon VPC (Virtual Private Cloud) for isolated and secure networking environments.
* Design multi-tier network topologies and route wide-area network traffic securely.

### Tasks to be carried out this week:

| Day | Task | Start Date | Completion Date | Reference Material |
| :--- | :--- | :--- | :--- | :--- |
| **2** | - Study core theoretical concepts of Amazon VPC: IP address space allocation and partitioning using CIDR blocks. | 05/25/2026 | 05/25/2026 | <https://cloudjourney.awsstudygroup.com/> |
| **3** | - Distinguish logical boundaries and roles of Public Subnets (Internet-facing zones) and Private Subnets (securely isolated data zones). | 05/26/2026 | 05/26/2026 | <https://cloudjourney.awsstudygroup.com/> |
| **4** | - Research Internet Gateways (IGW) and traffic direction mechanisms using Route Tables. | 05/27/2026 | 05/27/2026 | AWS VPC Console |
| **5** | - Explore NAT Gateway solutions for providing secure, outbound-only Internet access to instances located in Private Subnets. | 05/28/2026 | 05/28/2026 | <https://cloudjourney.awsstudygroup.com/> |
| **6** | - Examine multi-layer firewall systems for network protection: Network ACLs (subnet-level stateless protection) and Security Groups (instance-level stateful protection). | 05/29/2026 | 05/29/2026 | <https://cloudjourney.awsstudygroup.com/> |

### Week 4 Achievements:
* Acquired a professional mindset in designing enterprise cloud network infrastructures (VPC Core Topology).
* Mastered methods for securely segmenting network traffic into public-facing and private data tiers.
* Mastered traffic routing mechanisms, gateway integration, and the deployment of multi-layer firewall defenses to safeguard the cloud environment.
